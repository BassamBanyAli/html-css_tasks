SEO Meta tags

SEO meta tags :are part of HTML code that provide information about a webpage to

search engines and website visitors.

These meta tags are inserted into the head section of an HTML document and they are not visible on the actual webpage.

They play an important role in search engine optimization (SEO)
by helping search engines understand the content and context of a webpage.

type of meta tags : 1-Meta Title
specifies the title of a webpage.
It is displayed as the title in search engine results and in the browser's title bar.
should be short clear and include relevent keywords

example :

2-Meta Description provides a brief summary of the webpage content example :
3-Meta Robots: This tag tells search engines how to treat your page
. "Index" means it can be included in search results,
"nofollow" means search engines shouldn't follow links on the page
"noindex" means it shouldn't appear in search results, and
"noarchive" means search engines shouldn't store a cached copy.
4-Viewport : The viewport meta tag controls the layout and scaling
of the webpage on different devices.
It is crucial for responsive web design and mobile optimization.
5-Canonical Link: This tag tells search engines the main URL for
a page when there are multiple URLs for the same content.
It helps avoid duplicate content problems and improves how search engines rank the page.

i-Frame

The HTML iframe tag specifies an inline frame.
An inline frame is used to embed another document within the current HTML document.

The div

The (div) element is by default a block element, meaning that it takes all available width, and comes with line breaks before and after.
The (div) element has no required attributes, but style, class and id are common.
(div) as a container : The (div) element is often used to group sections of a web page together.
